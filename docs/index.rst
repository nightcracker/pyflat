pyflat documentation
====================

pyflat is a library for the creation of 2D interactive applications such as games and animations.

It aims to abstract away platform specific details, such as window opening and sound playback, and provide a single consistent interface to those features. On top of that it tries makes your life easier by including functions that can be found in nearly every 2D application, such as image loading, drawing primitives and font rendering.

pyflat is Open Source and released under the BSD 2-Clause license.
